# Media Scanner

Периодически обходит указанный каталог, находит аудио-,видеофайлы и изображения и сохраняет их список в `.media_files` в формате JSON.

## Требования
- Linux
- g++, C++17
- Make

## Сборка
```
make
```

## Запуск
```
./scanner [interval] [path]
```

- interval — интервал сканирования в секундах (по умолчанию 30)
- path — сканируемый каталог (по умолчанию $HOME)

## Результат
Файл .media_files создаётся в корне сканируемого каталога.

Формат:
```
{
  "audio": ["path_to_file.ext"],
  "video": ["path_to_file.ext"],
  "images": ["path_to_file.ext"]
}
```
